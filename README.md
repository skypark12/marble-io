## 확장으로 사용

이 저장소는 MakeCode에서 **확장**으로 추가될 수 있습니다.

* [https://makecode.microbit.org/](https://makecode.microbit.org/) 열기
* **새로운 프로젝트**에서 클릭
* 톱니바퀴 모양 메뉴에서 **확장**을 클릭합니다
* **https://github.com/skypark12/marble-io**으로 검색하고 가져오기

## 이 프로젝트 편집 ![상태 배지 빌드](https://github.com/skypark12/marble-io/workflows/MakeCode/badge.svg)

MakeCode에서 이 저장소를 편집합니다.

* [https://makecode.microbit.org/](https://makecode.microbit.org/) 열기
* **가져오기**를 클릭한 다음 **가져오기 URL**를 클릭합니다
* **https://github.com/skypark12/marble-io**를 붙여넣고 가져오기를 클릭하세요.
